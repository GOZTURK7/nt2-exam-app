## 1. Uygulamaya Genel Bakış

Bu uygulama, Hollanda'da yapılan **NT2 Staatsexamen** sınavına odaklı, günlük ve modüler bir çalışma programı sunar. Her gün belirli bir konu üzerinde — kelimeler, kalıplar ve sınav pratiği — yapılandırılmış şekilde çalışırsınız.

### Temel kavramlar

| Kavram | Açıklama |
|--------|----------|
| **Program Günü** | Tek bir konuyu kapsayan günlük çalışma birimi |
| **Stres Kelimesi ⚡** | Sınav stresinde çok unutulan somut kelime |
| **Bilmiyorum ×** | Kendinizin işaretlediği, tekrar gerektiren kelime |
| **Sınav Simülasyonu** | Gerçek süre ve mikrofon kaydıyla sınav pratiği |

### Desteklenen sınav bölümleri

```
Spreken   → Konuşma (Deel 1 · Deel 2 · Deel 3)
Schrijven → Yazma
Lezen     → Okuma
Luisteren → Dinleme
```

> Şu anda **B2 · Spreken** modülü aktiftir. Diğer modüller ilerleyen güncellemelerde eklenecektir.

## 2. Ana Ekran

Uygulamayı açtığınızda karşınıza gelen ekrandır.

```
┌─────────────────────────────────────────┐
│  NT2 · PREP          [B2]   [TR]        │  ← Üst Bar
├─────────────────────────────────────────┤
│  NT2 Sınav Hazırlığı     Sınava         │
│  Program II – B2          33            │
│  B2 seviyesinde...       gün kaldı      │
│ ─────────────────────────────────────── │
│  [🎤 Spreken ▼]              0 / 1      │  ← Aktif modül
│                           İlerleme      │
│ ┌───────────────────────────────────┐   │
│ │  GÜN 1   [DEEL 2 – PLAATJES]  ○  │   │  ← Gün kartı
│ │  Giysi, Aksesuar & Beden         │   │
│ │  📖 23  💬 6  ⏱ 20s  ⚡ 17      │   │
│ └───────────────────────────────────┘   │
│ ─────────────────────────────────────── │
│  0 Gün Serisi  0 Tamamlanan  0 Kelime  │  ← İstatistikler
├─────────────────────────────────────────┤
│  🎤 Konuşma  ✏️ Yazma  📖 Okuma  🎧 Dinleme │  ← Alt Navigasyon
└─────────────────────────────────────────┘
```

### Üst Bar

| Öğe | İşlev |
|-----|-------|
| **`B1` / `B2` butonu** | Aktif program seviyesini değiştirir. B1 veya B2 seçin. |
| **`TR` / `EN` butonu** | Uygulama dilini Türkçe ↔ İngilizce olarak değiştirir. Tüm çeviriler ve açıklamalar seçtiğiniz dile göre gösterilir. |

### Gün Kartı

Her kart bir günlük çalışma birimini temsil eder. Kartın üzerindeki bilgiler:

| Simge | Anlam |
|-------|-------|
| `📖 23` | O günün kelime sayısı |
| `💬 6` | O günün kalıp cümle sayısı |
| `⏱ 20s` | Sınav görevinin süresi (saniye) |
| `⚡ 17` | Stres kelimesi sayısı (dikkat gerektiren somut kelimeler) |
| `○` (boş daire) | Gün henüz tamamlanmadı |
| `✓` (sarı tik) | Gün tamamlandı |

**Karta tıklayarak** çalışma ekranına geçersiniz.  
**Dairedeki tikine tıklayarak** günü tamamlandı / tamamlanmadı olarak işaretleyebilirsiniz (karta girmeye gerek yok).

## 3. Çalışma Ekranı

Bir gün kartına tıkladığınızda açılan ekrandır. İki sekme içerir:

```
┌─────────────────────────────────────────┐
│  ← GÜN 1                               │  ← Geri butonu + başlık
│    Giysi, Aksesuar & Beden             │
│ ─────────────────────────────────────── │
│  [ Kelimeler ]   [ Sınav Simülasyonu ]  │  ← Sekme çubuğu
├─────────────────────────────────────────┤
│                                         │
│          (sekme içeriği)                │
│                                         │
└─────────────────────────────────────────┘
```

**Geri** butonuna (←) tıklayarak Ana Ekran'a dönersiniz. İlerlemeniz kaybolmaz.

## 4. Kelimeler Sekmesi

### 4.1 Kelime Kartları

Her kelime için bir kart gösterilir:

```
┌──────────────────────────────────────────────┐
│  de veter / de veters              [  ✓  ]   │
│  ayakkabı bağı / ayakkabı bağları            │
│ ──────────────────────────────────────────── │
│  De veter is te lang.                        │
│  KATEGORİ: kleding                           │
└──────────────────────────────────────────────┘
```

```
╔══════════════════════════════════════════════╗
║  de maat              ⚡ Stres Kelimesi  [×] ║  ← Sarı neon = stres kelimesi
║  beden / ölçü                               ║
║ ──────────────────────────────────────────── ║
║  Welke maat heeft u?                        ║
║  KATEGORİ: kleding                          ║
╚══════════════════════════════════════════════╝
```

### 4.2 Stres Kelimeleri ⚡

**Sarı neon çerçeveli** kelimeler, sınav stresinde en çok unutulan **somut kelimelerdir** (giysinin parçaları, aksesuarlar gibi gözle görülebilir nesneler). Bu kelimelere özellikle dikkat edin.

### 4.3 Bilmiyorum Butonu

Her kartın sağ üstünde küçük bir buton vardır:

| Durum | Görünüm | Anlam |
|-------|---------|-------|
| Biliyorum | `✓` (yeşil kenarlık) | Kelimeyi biliyorsunuz |
| Bilmiyorum | `×` (turuncu kenarlık) | Kelimeyi tekrar etmeniz gerekiyor |

Butona tıkladığınızda kelime, **tekrar listesine** eklenir veya listeden çıkarılır. Bu liste ilerleyen güncellemelerde size özel tekrar seansları oluşturmak için kullanılacaktır.

### 4.4 Kalıp Cümleler

Kelime kartlarının altında, o gün için hazırlanmış **sınav kalıpları** listelenir. Örnek:

> _Zoals u kunt zien op het plaatje..._  
> Resimde gördüğünüz gibi...

Bu kalıpları ezberleyip sınav konuşmanızda kullanabilirsiniz.

## 5. Sınav Simülasyonu Sekmesi

Gerçek sınav koşullarını simüle eder: **belirli bir süre** içinde **mikrofona** konuşursunuz, ardından kaydınızı dinleyebilirsiniz.

### 5.1 Ekran Düzeni

```
┌─────────────────────────────────────────┐
│  SINAV GÖREVİ                           │
│  Resimdeki kişinin ne yaptığını 20      │
│  saniye içinde tarif edin...            │
├─────────────────────────────────────────┤
│                                         │
│              ╭─────────╮               │
│            ╭─┤  20 sn  ├─╮             │  ← Dairesel sayaç
│            │ ╰─────────╯ │             │
│            ╰─────────────╯             │
│                                         │
│         [ 🎤  Kaydet ]                  │  ← Başlat butonu
└─────────────────────────────────────────┘
```

### 5.2 Kayıt Akışı

```
[Kaydet] tıkla
    │
    ▼
Tarayıcı mikrofon izni ister
    │
    ├─ İzin verilmedi → Hata mesajı gösterilir
    │
    └─ İzin verildi → Sayaç geri saymaya başlar (sarı renk)
         │
         ├─ Süre bitince → Kayıt otomatik durur
         │
         └─ [■ Durdur] butonuyla erken bitirebilirsiniz
              │
              ▼
         Kayıt tamamlandı (yeşil "Bitti")
              │
              ▼
         Ses oynatıcı görünür → Dinleyin
              │
         ┌────┴────┐
         │         │
     [Sil]     [Tekrar Kaydet]
```

### 5.3 Sayaç Renkleri

| Renk | Anlam |
|------|-------|
| Mavi | Hazır, kayıt başlamadı |
| Sarı (parlayan) | Kayıt devam ediyor |
| Yeşil | Tamamlandı |

### 5.4 Mikrofon İzni

İlk kullanımda tarayıcı mikrofon erişimi sorar. **İzin Verin** — aksi hâlde kayıt çalışmaz. Tarayıcı ayarlarından sonradan da izin verebilirsiniz:

- **Chrome:** Adres çubuğundaki kilit simgesi → Mikrofon → İzin Ver
- **Firefox:** Adres çubuğundaki kilit simgesi → Daha Fazla Bilgi → İzinler

## 6. İlerleme Takibi

Tüm ilerlemeniz **tarayıcınızın yerel belleğine** (localStorage) kaydedilir. Uygulamayı kapatsanız bile verileriniz korunur. Ancak tarayıcı geçmişini temizlerseniz ilerleme sıfırlanabilir.

### Ne kaydedilir?

- Tamamlanan günler (✓ işaretli kartlar)
- Bilmiyorum olarak işaretlenen kelimeler
- Aktif seviye (B1 / B2)

### İstatistik şeridi

Ana ekranın altındaki üç kutu:

| Kutu | Gösterilen |
|------|-----------|
| **Gün Serisi** | Arka arkaya çalışılan gün sayısı *(yakında aktif)* |
| **Tamamlanan** | Toplam tamamlanan gün sayısı |
| **Kelime** | Tekrar listenizdeki kelime sayısı *(yakında aktif)* |

## 7. Stres Kelimeleri Sistemi

NT2 sınavında yaşanan en yaygın sorun, stres altında **somut kelimeleri unutmaktır** — bir kıyafetin parçasını, aksesuar adını, ya da gündelik bir eşyayı tam olarak söyleyememek.

Bu uygulama, her günün kelimelerini iki gruba ayırır:

```
⚡ Stres Kelimesi (isConcreteWord: true)
   → Gözle görülebilen, somut nesneler
   → Sınav stresinde en çok unutulan
   → Sarı neon rengiyle vurgulanır

   Örnekler: de veter, de riem, de rits,
             de armband, de sjaal...

── Normal Kelime (isConcreteWord: false)
   → Soyut kavramlar, fiiller, kalıplar
   → Beyaz renkte gösterilir

   Örnekler: oprapen, dragen, de maat...
```

**Öneri:** Çalışma seansınıza stres kelimelerini ezberlemeyle başlayın. Bunlar az ama kritiktir.

## 8. Önerilen Çalışma Düzeni

Her program günü için aşağıdaki sırayı izlemenizi öneririz:

### Adım 1 — Kelimeleri Tara (5 dk)
Kelimeler sekmesini açın. Tüm kartları hızlıca gözden geçirin. Hiç bilmediğiniz kelimelerin üstüne `×` basın.

### Adım 2 — Stres Kelimelerine Odaklan (5 dk)
⚡ sarı kartları sesli tekrarlayın. Hollandaca kelimeyi söyleyin, ardından örnek cümleyi okuyun.

### Adım 3 — Kalıpları Ezberle (3 dk)
Sayfanın altındaki kalıp cümlelerini sesli okuyun. Bunları sınavda "açılış cümlesi" olarak kullanacaksınız.

### Adım 4 — Sınav Simülasyonu (5 dk)
Sınav Simülasyonu sekmesine geçin. Görevi okuyun ve **gerçekten konuşarak** kaydı başlatın. Bittikten sonra kaydınızı dinleyin:
- Duraklamalar var mıydı?
- Kelimeleri doğru telaffuz ettiniz mi?
- Kalıp cümlelerden birini kullanabildınız mı?

### Adım 5 — Güne Tik At ✓
Ana ekrana dönün, kartın üstündeki daireye tıklayarak günü tamamlandı olarak işaretleyin.

## 9. Sınav Formatı Hakkında

### NT2 Staatsexamen — Spreken Bölümü

Spreken sınavı üç bölümden oluşur:

| Bölüm | Ad | Süre | Açıklama |
|-------|----|------|----------|
| **Deel 1** | Korte antwoorden | ~1 dk | Kısa sorulara kısa yanıtlar |
| **Deel 2** | Plaatjes beschrijven | 20 sn × 3 | Resimlerdeki durumu anlatmak |
| **Deel 3** | Grafiek & tabel | ~2 dk | Grafik veya tablo yorumlamak |

Bu uygulama şu anda **Deel 2 – Plaatjes** için içerik sunmaktadır.

### Deel 2 için Taktikler

1. **Açılış cümlesi** kullanın:  
   _"Zoals u kunt zien op het plaatje..."_

2. **Kişiyi tanımlayın** — yaş, cinsiyet, giysi:  
   _"Er is een vrouw te zien. Ze draagt een blauwe sjaal en handschoenen."_

3. **Eylemi anlatın** (şimdiki zaman):  
   _"Ze is bezig met het oprapen van haar portemonnee."_

4. **Bitiriş cümlesi** ekleyin:  
   _"Dat is wat ik op het plaatje zie."_

5. **20 saniyeyi doldurun** — sessizlik puan kırar. Kelime bilemiyorsanız tanımlayın:  
   _"...een soort accessoire om haar nek..."_ (sjaal bilmiyorsanız)
